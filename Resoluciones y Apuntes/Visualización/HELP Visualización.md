
| Tipo de gráfico                     | Cuándo usarlo                                                                                | Ejemplo típico                                                     |
| ----------------------------------- | -------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| **Tabla**                           | Para mostrar datos exactos o detallados, cuando los valores numéricos son importantes.       | Ventas por mes y región (con cifras concretas).                    |
| **Barras / Columnas**               | Para **comparar valores** entre categorías o grupos.                                         | Comparar suscripciones por región o producto.                      |
| **Líneas**                          | Para mostrar la **evolución en el tiempo** (tendencias).                                     | Crecimiento de clientes en los últimos 5 años.                     |
| **Torta (Pie chart)**               | Para mostrar **porcentajes o proporciones** de un total.                                     | Porcentaje de clientes por región.                                 |
| **Dispersión (Scatter)**            | Para ver la **relación entre dos variables numéricas**.                                      | Relación entre edad y monto de compra.                             |
| **Burbujas**                        | Igual que dispersión, pero el **tamaño de la burbuja agrega una tercera variable**.          | Ventas (eje X), ganancias (eje Y), cantidad de empleados (tamaño). |
| **Treemap**                         | Para mostrar **jerarquías o proporciones** dentro de categorías anidadas.                    | Ventas por categoría y subcategoría.                               |
| **Nube de palabras**                | Para representar la **frecuencia de palabras o temas**.                                      | Palabras más usadas en reseñas de clientes.                        |
| **Infograma**                       | Para **comunicar datos con diseño visual y texto**, de forma resumida y atractiva.           | Mostrar resultados clave de una encuesta.                          |
| **Mapa de símbolos proporcionales** | Para **mostrar datos geográficos**, donde el **tamaño del símbolo** representa una magnitud. | Población por ciudad.                                              |
| **Mapa de área (coroplético)**      | Para **mostrar variaciones por zona geográfica** (coloreando regiones según un valor).       | Nivel de contaminación por provincia.                              |
**Tip rápido para elegir:**

- Si hay **categorías → barras**.
- Si hay **tiempo → líneas**.
- Si hay **porcentajes → torta o treemap**
- Si hay **coordenadas → mapa o dispersión**.
- Si hay **texto → nube de palabras**.
